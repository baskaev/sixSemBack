Weather Station Project (Windows Edition, FINAL v2)

1. Убедись что у тебя установлен Docker Desktop для Windows
2. Распакуй этот архив в любую папку
3. Дважды кликни на файл run.bat — сборка запустится автоматически
4. После запуска открой в браузере: http://localhost:8088/

Backend API будет доступен на: http://localhost:8000/

MySQL проброшен на порт: 33061 (чтобы не конфликтовать с локальной базой)

Если будут вопросы — пиши разработчику :)

Удачи!
